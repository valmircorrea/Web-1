1ª atividade:
    Você deve construir a home page de um sistema de vendas de produtos/serviços bem simples.

    Nesta primeira atividade, o sistema terá duas páginas. Novas páginas serão colocadas depois.
    Na página inicial, deve existir o nome e a logo marca da loja no topo da página. Logo abaixo
    deve ter uma lista com as opções:

    - entrar/cadastrar

    - nossos produtos

    - buscar produtos

    - sobre nós

    Logo abaixo, coloque um video que descreva que apresenta alguns produtos. Mais abaixo,
    um tabela com pelo menos 6 imagens de produtos. Coloque uma âncora que permita ver a tabela
    quando o usuário clicar "nossos produtos". Pegue imagens e videos da Web

    Crie uma outra página que descreva informações sobre a empresa e coloque um link para "sobre nós".
    Use a mesma formatação da página inicial.

2º atividade
    Acrescente ao exercício anterior duas páginas: uma para cadastro dos clientes e outra para login. 
    Os dados do cadastro ficam ao seu critério, podendo incluir nome, sobrenome, senha, email, telefone,
    cpf, RG, etc.  Para o login o usuário deve definir uma senha no cadastro. Ajuste as páginas do 
    exercício anterior para incluir no menu as opções de login e cadastro.
    
3º atividade
    Aplique formatação CSS ao Sistema de Vendas que está sendo desenvolvido nas aulas práticas 
    anteriores (4 páginas - home, sobre nós, cadastro, login). A proposta é ter um layout com cabeçalho, 
    rodapé, menu e conteúdo. Vejam exemplo no W3school com ponto de 
    partida: https://www.w3schools.com/css/css_website_layout.asp
